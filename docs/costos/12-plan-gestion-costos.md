# 12. Plan de Gestión de Costos

- **Moneda Base:** CLP (pesos chilenos) para evaluación académica. USD para gastos reales de infraestructura.
- **Técnicas de Estimación:** Análoga (para hardware ESP32 basados en MercadoLibre) y Paramétrica (horas hombre simuladas).
- **Umbrales de Control:** Alerta inmediata si el consumo de AWS/Hetzner supera los USD $25/mes.
- **Reportes:** Revisión mensual de facturación cloud.
