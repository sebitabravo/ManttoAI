# 09. Plan de Gestión del Cronograma

- **Metodología:** Ágil-cascada híbrida (Sprints de 2 semanas para ejecución, cascada para diseño PMBOK).
- **Herramientas:** GitHub Projects (Kanban), Discord (Daily syncs informales).
- **Medición de Avance:** Commits integrados a `main` y PRs cerrados.
- **Protocolo de Desvíos:** Si un componente (ej. modelo ML) retrasa la cadena > 2 días, se activa un plan de contingencia (ej. usar data mockeada o simulador para desbloquear al frontend).
