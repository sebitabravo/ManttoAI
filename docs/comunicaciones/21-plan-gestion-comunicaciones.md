# 21. Plan de Gestión de las Comunicaciones

- **Canal Principal Interno:** Servidor de Discord (Diario, asíncrono).
- **Repositorio de Información:** GitHub Repository (`main` branch para código, carpeta `docs/` para PMBOK y arquitectura).
- **Notificaciones Automáticas:** GitHub Actions reporta el estado de los tests (Verde/Rojo) directamente en los PRs.
- **Reportes Académicos:** Entregables formales exportados a PDF desde Markdown para revisión de INACAP.
