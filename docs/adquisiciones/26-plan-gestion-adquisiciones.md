# 26. Plan de Gestión de Adquisiciones

**Adquisiciones Clave del Proyecto:**
1. **Hardware IoT:** 3x ESP32 DevKit v1, 3x Sensores DHT22, 3x MPU-6050. Comprados vía MercadoLibre Chile para asegurar tiempos de entrega cortos.
2. **Infraestructura Cloud (prototipo):** VPS CX22 en Hetzner Cloud para desarrollo y demo. El plan de negocios proyecta migración a Digital Ocean (Droplet 2GB + PostgreSQL gestionada, São Paulo) para la operación comercial. Ver [`docs/costos/12-plan-gestion-costos.md`](../costos/12-plan-gestion-costos.md).

**Estado prototipo:** ✅ Hardware adquirido (3 kits ESP32 + sensores). VPS Hetzner aprovisionado con Dokploy. La adquisición de infraestructura Digital Ocean y los 6 kits ESP32 adicionales (2 por rubro) corresponde al Año 1 del plan de negocios.
